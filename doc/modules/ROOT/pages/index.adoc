= TextFoundry Documentation

TextFoundry documentation is published as an Antora site. The current structure
separates architectural intent, implementation status, and API reference.

== Source of Truth

The Antora pages in this module are thin wrappers around the existing source
documents in `doc/*.adoc`. This keeps the project on a single documentation
source while enabling Antora publication.

== Structured Space

The `Project Docs` section provides a hierarchical documentation space closer to
an internal knowledge base. It is intended as the publication layer for:

* project PRD and architecture materials
* structured AI / ML delivery artifacts
* process and operational documentation
* outputs produced through `spec kit` and `ML LLM Delivery Pipeline` skills
