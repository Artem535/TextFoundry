= 5. Расчёт трудозатрат, риски, ограничения

Страница зарезервирована под:

* оценки и допущения по delivery
* продуктовые и технические риски
* ограничения по времени, команде и инфраструктуре
* rollout constraints для AI / ML функциональности

При наполнении ML/LLM delivery-артефактами сюда логично переносить summary из
`Delivery Plan`, `Security Note` и `Evaluation Plan`.
