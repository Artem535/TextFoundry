= Block Generation

Функция отвечает за генерацию одного нового блока из пользовательской
инструкции и возврат reviewable draft в authoring workflow.

Текущее состояние:

* реализована в `tf::Engine` как explicit AI workflow
* использует `IBlockGenerator` и `OpenAiCompatibleBlockGenerator`
* доступна из GUI block editor через `BlockEditorVm.generate()` и `revise()`
* не вызывается автоматически из `Render()`
* не публикует block version без ручного save
