= Normalization

Функция отвечает за семантическую нормализацию текста и composition через
explicit AI call.

Текущее состояние:

* поддерживается `Normalize(...)`
* поддерживается `NormalizeComposition(...)`
* normalizer и block normalizer отделены от deterministic render path
* GUI expose-ит style controls для render и composition workflows
* normalized result является derived artifact, а не hidden mutation render path
