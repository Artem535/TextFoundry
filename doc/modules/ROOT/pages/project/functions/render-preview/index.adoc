= Render Preview

Функция отвечает за deterministic render, raw preview и optional normalized
preview поверх выбранной composition в GUI.

Текущее состояние:

* реализована в `RenderViewModel`, `RenderPage.qml`, `tf::Engine` и `Renderer`
* поддерживает три режима просмотра: `Raw`, `Rendered`, `Normalized`
* работает с published compositions и version-aware render path
* позволяет задавать runtime params и выбирать конкретную версию composition
* normalization вызывается только явно и не является частью базового render path
* поддерживает copy для rendered и raw output, а также reset/clear render state
