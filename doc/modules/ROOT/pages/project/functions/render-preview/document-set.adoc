= Render Preview Document Set

== 1. Покрытие набора документов

Покрытие функции сейчас разложено так:

* продуктовый intent - `PRD`
* системная архитектура - `System Design`
* ключевое решение по mode separation - `ADR`
* контракт preview workflow - `Service Spec`
* release quality expectations - `Evaluation Plan`
* rollout and ownership - `Delivery Plan`
* operations - `Runbook`
* data handling constraints - `Security Note`
* normalization-specific prompting note - `Prompt Template`
