= Render Preview Delivery Plan

== 1. Область поставки

В поставку capability входят:

* GUI render screen
* deterministic render orchestration
* raw preview path
* optional normalized preview
* version-aware composition preview

== 2. Текущий статус

Текущий статус:

* capability уже реализована в коде
* отсутствовал отдельный function-level document set
* ключевое следующее действие - закрепить terminology и test expectations

== 3. Рабочие потоки и владельцы

* GUI owner:
* page layout
* preview mode UX
* status texts and clipboard actions
* engine owner:
* render contract
* version loading
* deterministic rendering guarantees
* AI owner:
* normalized preview behavior
* normalizer availability and prompt policy

== 4. Карта зависимостей

* composition repository
* version metadata loading
* renderer and block cache
* session-level engine configuration
* optional normalizer configuration

== 5. Этапы rollout

* formalize docs for render-preview
* align QA checks with deterministic render contract
* add explicit regression set for params/version validation
* review raw preview helper against future renderer changes

== 6. Блокеры и риски

* raw preview helper duplicates part of render-like behavior
* user expectations may drift toward treating normalized output as canonical
* future prompt server work may create pressure to reuse GUI-specific semantics

== 7. Критерии готовности

* composition/version selection behavior documented
* deterministic render path documented as canonical runtime output
* normalization explicitly documented as optional derived preview
* stale-state behavior documented
* error surfaces documented
