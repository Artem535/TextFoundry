= Composition Rewrite

Функция отвечает за block-preserving rewrite workflow для composition.

Текущее состояние:

* поддерживается preview/apply модель
* отдельный rewriter подключается через engine
* GUI поддерживает `Rewrite Blocks` dialog с patch preview
* apply публикует новые block versions и новую composition version
* blind overwrite composition не поддерживается
