= Composition Rewrite Document Set

== 1. Покрытие набора документов

Для `Composition Rewrite` сейчас покрыты:

* `PRD`
* `ADR`
* `System Design`
* `Service Spec`
* `Evaluation Plan`
* `Delivery Plan`
* `Runbook`
* `Security Note`

== 2. Что уже согласовано между документами

Согласованы:

* preview/apply model;
* patch-oriented rewrite semantics;
* block-preserving behavior;
* placeholder-safe apply;
* distinction between suggestion and mutation.
