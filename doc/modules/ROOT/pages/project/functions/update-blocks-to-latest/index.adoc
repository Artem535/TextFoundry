= Update Blocks To Latest

Функция описывает controlled workflow обновления ссылок композиции на последние
доступные версии блоков без ручного редактирования каждого `BlockRef`.

Ключевые свойства capability:

* работает только по явному действию пользователя;
* читает выбранную published composition version;
* заменяет версии `BlockRef` на latest published block versions;
* публикует новую minor-версию композиции только если есть реальные изменения;
* не меняет текст блоков и не вызывает AI provider.
