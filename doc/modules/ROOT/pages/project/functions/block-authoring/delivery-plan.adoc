= Block Authoring Delivery Plan

== 1. Объём поставки

Функция включает:

* дерево и поиск блоков
* просмотр подробностей и версий
* вкладочный редактор
* публикацию новой версии
* снятие версии с использования
* удаление блока
* AI-черновик и AI-ревизию

== 2. Текущее состояние

* функция уже реализована в коде
* отсутствовал самостоятельный подробный набор документов
* следующая задача — удержать единый язык описания и критерии качества

== 3. Потоки работ и владельцы

* владелец интерфейса:
* дерево, поиск, вкладки, предупреждения о несохранённых изменениях
* владелец ядра:
* публикация, обновление версий, удаление, ограничения ссылочной целостности
* владелец AI-слоя:
* качество генерации и ревизии

== 4. Карта зависимостей

* локальное хранилище блоков
* репозиторий композиций для проверки ограничений удаления
* модель дерева блоков
* редактор вкладок
* необязательный AI-провайдер

== 5. Шаги внедрения

* формализовать документацию функции
* согласовать терминологию версионирования
* закрепить регрессионные сценарии на публикацию и удаление
* уточнить ожидания от AI-черновика и AI-ревизии

== 6. Блокеры и риски

* сложное состояние вкладок
* смешение ручного авторинга и AI-подсказки в восприятии пользователя
* риск неправильного удаления при изменении логики ссылок

== 7. Контрольный список готовности

* правила новой версии задокументированы
* правила удаления задокументированы
* AI-режимы описаны как вспомогательные
* вкладочный режим и несохранённые изменения описаны
