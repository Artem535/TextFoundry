= Prompt Slicing Security Note

== 1. Data Sensitivity

Source prompt может содержать целые рабочие инструкции, внутренние процессы и
чувствительные формулировки.

== 2. Key Risks

Риски:

* source prompt может содержать конфиденциальный контент
* provider получает полный raw input
* generated decomposition может раскрывать лишнюю структуру

== 3. Storage and Retention

* raw source prompt пересекает provider boundary
* retention policy must be reviewed per provider

== 4. Required Controls

Меры:

* классифицировать input sensitivity
* использовать review-before-apply
