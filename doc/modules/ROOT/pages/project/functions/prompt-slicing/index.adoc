= Prompt Slicing

Функция отвечает за декомпозицию исходного prompt или большого текста в набор
предлагаемых блоков.

Текущее состояние:

* реализована через `PromptSlicingRequest`
* использует batch generation path
* доступна из GUI как `BlockSliceDialog`
* поддерживает create-batch и update-composition режимы
* publish выполняется только после preview и явного `Publish All`
