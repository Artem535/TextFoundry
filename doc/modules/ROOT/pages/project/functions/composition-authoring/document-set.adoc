= Composition Authoring Document Set

== 1. Сводка покрытия

Набор документов покрывает:

* продуктовую цель работы с композициями
* архитектуру редактора структуры
* ключевое решение о работе через фрагменты и новые версии
* контракт данных и действий пользователя
* операционные и защитные ограничения

== 2. Сквозные пробелы

Что ещё можно усилить:

* расширенный набор контрольных сценариев на перестановку фрагментов
* отдельные проверки на сложные композиции с большим числом ссылок
* более плотную увязку с будущими документами по сопровождению композиций

== 3. Следующие обновления

* связать набор с будущим набором по `composition-maintenance`
* уточнить словарь терминов по видам фрагментов
* синхронизировать с документами нормализации и переписывания
