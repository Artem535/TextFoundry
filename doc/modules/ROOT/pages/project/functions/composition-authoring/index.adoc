= Composition Authoring

Функция отвечает за создание, просмотр, редактирование, версионирование и
снятие с использования композиций в настольном интерфейсе `TextFoundry`.

Текущее состояние:

* реализована в `CompositionsViewModel`, `CompositionEditorViewModel`,
  `CompositionsPage.qml`, `CompositionEditorWindow.qml` и `tf::Engine`
* поддерживает список композиций, поиск, список версий и подробности выбранной
  композиции
* использует отдельный редактор композиции с боковой панелью и рабочей
  областью фрагментов
* позволяет собирать композицию из ссылок на блоки, статического текста и
  разделителей
* поддерживает перестановку фрагментов, вставку, вырезание, вставку из буфера и
  очистку разделителей
* сохраняет композицию как новую опубликованную версию, а не перезаписывает
  старую
* не выполняет автоматическое изменение опубликованной композиции без явного
  действия пользователя
