= Composition Authoring Prompt Template

== 1. Назначение

Для базового авторинга композиции отдельный шаблон промпта не нужен.

Функция по своей природе является структурным редактором, а не AI-сервисом.
Этот документ сохраняется для единообразия набора pipeline-документов.

== 2. Ограничение

Если в будущем появятся AI-помощники именно для авторинга композиции, они
должны:

* работать поверх модели фрагментов, а не вместо неё
* не публиковать композицию автоматически
* не подменять собой базовый редактор структуры

== 3. Контракт результата

* результат AI-помощника, если он появится, должен быть переводим в набор
  фрагментов
* результат не должен считаться опубликованной композицией без ручного
  подтверждения
