# ML / LLM Security Note

Read:

* `../../../ML LLM Delivery Pipeline/08 Security Note Template.md`

Use this skill for security posture and compliance constraints around ML/LLM
systems.

## Cover

* data sensitivity
* storage and retention
* access model
* logging restrictions
* vendor and compliance constraints
* security controls
* risks and mitigations

## Guardrails

* Be explicit about what must not be logged or sent to vendors.
* Mention retention and deletion, not just encryption.
