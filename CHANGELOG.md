# Changelog

All notable changes to TextFoundry are documented in this file.

## 0.2.3 - 2026-04-13

### Added
- Composition-level `Rewrite Blocks` workflow with AI-generated block patch preview and apply flow.
- Raw prompt compare for composition versions with side-by-side diff view.

### Changed
- Improved compare dialog layout with line-based rendering, wrapping, and line numbers.
- Refined composition block rewrite dialog sizing and instruction input behavior.

### Fixed
- Fixed scrolling and rendering issues in raw prompt compare.
- Fixed clipping and usability issues in the `Rewrite Blocks` instruction field.

## 0.2.2 - 2026-04-08

### Added
- Composition block rewrite engine pipeline and OpenAI-compatible adapter.
- Compare improvements for composition raw prompts.

### Changed
- Updated prompts and rewrite flow foundations for block-preserving edits.

## 0.2.1 - 2026-04-08

### Added
- Workspace UX improvements for blocks and compositions.
- Safer editor flow with unsaved-changes warnings.

### Changed
- Improved multi-tab editing and action placement across the GUI.

## 0.2.0 - 2026-04-01

### Added
- Versioned composition editing and revision comments.
- RPM packaging pipeline and release packaging support.

### Changed
- Refactored versioned GUI workspaces for blocks, compositions, and render flows.
